requires GCC 13.1.0 MinGW for building on Windows

Linux:
sfml can be installed using the fillowing command
sudo apt-get install libsfml-dev


To build an excutable file with static link libraries (no need for .dll)

compiling
g++ -c main.cpp -I".\src\include" -DSFML_STATIC

linking
g++ main.o -o gem_puzzle -L".\src\lib" -lsfml-graphics-s -lsfml-audio-s -lsfml-window-s -lsfml-system-s -lopengl32 -lfreetype -lwinmm -lgdi32 -mwindows

running
.\gem_puzzle


To build an excutable file with dynamic link libraries (.dll need to be in the same directory as the excutable)

compiling
g++ -c main.cpp -I".\src\include"

linking
g++ main.o -o gem_puzzle -L".\src\lib" -lsfml-graphics -lsfml-audio-s -lsfml-window  -lsfml-system

running
.\gem_puzzle
